About the Project: The goal of this project was to show the UI/UX process for the game Overwatch 2. This project started with the player journey and ended with UI Mockups and a Colorblindness test.

Roles and Responsibilities: The role in this project was to recreate the UI/UX design for Overwatch.Some responsibilities included designing Wireframes, Usability Tests and Player Profiles. Create a UI styleguide along with UI Mockups for the game. Design Wireframes & assets that match the style of the game.

Length of the Project: 8 weeks.

Challenges: Making changes that fit the needs of all players. Recreating styles and assets.
